Best Practices

•Point links to the resource's publicly available abstract, not behind the paywall.
•Review all fields that citation managers automatically fill.
•Researchers should include the access date in their references.
•The date clarifies which version of the article the researcher referenced.
•Use archiving and snapshot systems like Memento and Wayback Machine.
•Cite both the original link and a persistent link.
•Notify readers if a publication has been retracted or changed.
•Allow users to revert to the original link if the persistent one is less maintained.
•Regularly check metadata to keep all their links active and accurate.
•Regularly run a link validator on every website and publication.
•Avoid 3rd-Party DOI services that create unreliability.
